# hitokoto-MDUI

# 如何使用？
1.下载本仓库的所有文件
（您可以点击 <a href='https://github.com/Tvogmbh/hitokoto-MDUI/releases/tag/hitokoto-MDUI'>Downlaod hitokoto-MDUI</a> 下载本仓库的所有文件）

2.点击idnex.html文件可以直接在本地运行，无需搭建。
